# Weiss Cube — Build Plan (v2: Toyo-model pivot, 2026-06-15)

## What changed (pivot decided 2026-06-15)
Moving toward the **Toyo University club model** (ref blog: toyotakujogame.hatenablog.com/entry/2019/10/14/231017).
Goal: a **simpler v1 with far fewer unique cards**, using **real modern cards** instead of heavily reworked customs.

### The three scoping decisions
1. **Custom cards — minimal, baked-in universal rule.** We DO still print custom cards, but the ONLY change baked in is: **every card gets all traits + responds to all name references** (the club's blanket "all traits / all names" rule, printed onto the card instead of enforced verbally). Cards otherwise stay as printed and balanced. Baking it into the card (vs. a pure house rule) is deliberate: a **future version can strip traits back on select cards to introduce real traited archetypes.** Color may still be reassigned to balance the 50-per-color split. *(Allowed change scope = traits, color, card references — unchanged from project rules.)*
2. **Card pool — best cards, era-agnostic, lean modern.** Pick the cleanest cards for the environment regardless of set; prefer recent printings where equal.
3. **Color identity — light themes.** Loose leanings per color, not enforced. Mostly good-stuff + a CX combo or two per color.

## Reference model (Toyo club)
- **4 players → 200 physical cards, 50 per color.** Heavy duplication ⇒ few distinct cards (the "less unique cards" win).
- **Per-color physical distribution:** Lv0 **18** · Lv1 **10** · Lv2 **8** · Lv3 **10** · Event **4** = 50.
- **Copy caps:** max **4** per card; **Lv1 CX combos = 6** (so a contested combo still leaves ~3 each to two drafters).
- **Inclusion rule:** every card must function standalone — the ONLY exception is CX combos.
- **Given to all (not drafted), 8 cards:** brainstorm/集中 ×4 + a Filres-compatible card in **each color** ×1. Guarantees a consistency engine regardless of pick luck.
- **Climaxes taken separately after the draft** (8 per deck), like MTG basic lands. Proxy to match trigger icons if short.
- **Deckbuild:** 50 picked + 8 given + 8 CX = 66 → cut 16 → **50-card deck**.
- **Draft flow:** pick until you hold 50; take 6–10 per pass (8×4 or 6×3 suggested); no side change.
